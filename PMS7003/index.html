Connect PMS7003 sensor directly to UART port or via USB adapter.

    sensor module  10 pin connector
|----------------|   pin    n   G7
||----|          | (pin 8) N/C red ------   not used
||1  2|          | (pin 7) N/C black ----   not used
||.  .|          | (pin 6) res yellow - left unused (or -> 10K Ohm -> 3V3 pin)
||9 10|          | (pin 5) Tx  green --------> Rx -----------\ USB/PI UART
||----|          | (pin 4) Rx  blew  --------> Tx -----------\ USB/PI UART
|                | (pin 3) set white ---- left unused (or -> 10K Ohm -> 3V3 pin)
|                | (pin 2) Gnd orange --------> Gnd-----------/ USB/PI
|                | (pin 1) 5V  purple --------> 5V -----------/ USB/PI
|----------------|

If connected directly to pi UART, then enable hadrware serial port and disable console:

sudo raspi-config

[5] Interfacing Options > Serial section, disable the serial login shell and enable the serial interface. After reboot your active serial port probably is '/dev/serial0'

If connected via USB adapter then your serial port is '/dev/ttyUSB0'

Sensor have two modes active and passive: - in active mode fan spins all the time and single read takes about 2s - in passive mode sensor wakeup only for read, fan start spin before reading and stops after, single read takes about 40s 

sudo pip install pyserial

Enable and configure sensor in config file
